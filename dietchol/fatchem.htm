Chemistry of Cholesterol and Fatty Acids


Return to the Impact of Dietary Fats on Cholesterol"

    Cholesterol and fatty acids are amphiphilic molecules with a polar group at one end and a lipophilic side chain.


    Figure 1 (click to show large image) shows the chemical structure of cholesterol and three fatty acids. Cholesterol structure is based on a steroid nucleus with a very lipophilic side chain in the 17beta-position. A 3beta-hydroxyl group forms the polar end of cholesterol. Fatty acids have a polar carboxylic acid group at one end and a long lipophilic hydrocarbon chain at the other end. Fatty acids are classified by the number of carbon atoms in the lipophilic chain and the number of double bonds found along the chain. Introduction of a double bond creates a degee of unsaturation (saturation refers to being completely filled with hydrogen). Palmitic acid (from coconut oil) is a 16 carbon completely saturated fatty acid. Oleic acid (from olive oid) is an 18-carbon monounsaturated fatty acid and linoleic acid (from fish oils) is an 18-carbon polyunsaturated fatty acid.

    These structures can be edited in 2-dimensions by installing and helper application called "ISISDraw.exe" and can be viewed in 3-dimensions by installing a helper application call "Raswin.exe". You can download the freeware program ISISDraw.exe (by Molecular Design Ltd.) and the freeware program Raswin (by Roger Sayle, Glaxo Inc.) from a variety of anonymous FTP sites. After these files are installed, you can configure your Web browser to recognize certain file type by using the following directions (these directions refer to NetScape version 2.0):

    1. Select the drop down menu "Options - General Preferences" and click on the "Helpers" tab.
    2. Press the button labeled "Create New Type..." and enter Mime Type: application Mime Sub Type: pdb.
    3. Set the File Extensions: = pdb
    4. Select the Radio Button "Launch Application" and then Browse for the location of "Raswin.exe".
    5. Now you should see the icon for Raswin. You have now linked all .PDB (Brookhaven Protein DataBank files) to the Helper Viewer application .PDB
    6. Follow the same procedure for linking .ENT (Brookhaven Entry) files to Raswin.exe
    7. Follow the same procedure for linking .TGF (2D MDL Graphics Files) to ISISDraw.exe.
